[b]SimpleAds[/b]

This addon allows you to display HTML code in various locations of forum. You can limit ads by
[list]
[li]time[/li]
[li]click[/li]
[li]impressions[/li]
[li]membergroups[/li]
[li]actions[/li]
[/list]
You can also create custom ad positions.

The addon requires no template or source edits.
